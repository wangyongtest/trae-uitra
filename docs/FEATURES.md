## 缓存命中率监控

Reasonix 架构核心特性，实时监控和优化 Prompt Cache 使用效率。

| 功能 | 状态 | 说明 |
|------|------|------|
| 实时命中率显示 | ✅ | 对话过程中实时展示当前会话缓存命中率 |
| 三色状态灯 | ✅ | 🟢 绿色（≥90%）/ 🟡 黄色（70-90%）/ 🔴 红色（<70%） |
| 历史趋势图 | 🔄 | 缓存命中率随时间变化的折线图 |
| Token 节省统计 | ✅ | 展示因缓存命中节省的 Token 数量和费用 |
| 诊断建议 | ✅ | 命中率低时自动给出优化建议 |
| 前缀锁定状态 | 🔄 | 显示当前前缀锁定是否生效 |
| 缓存层详情 | 📋 | 展示五层上下文各层的缓存贡献 |
| 会话间缓存对比 | 📋 | 对比不同会话的缓存效率 |
| 命中率目标设置 | ✅ | 可配置目标命中率阈值（默认 90%） |
| 缓存事件日志 | 📋 | 记录缓存命中/未命中的详细事件 |

## 上下文管理

Reasonix 五层上下文架构的实现与管理。

| 功能 | 状态 | 说明 |
|------|------|------|
| 五层架构说明 | ✅ | 文档完整描述五层上下文设计 |
| 上下文使用量可视化 | 🔄 | 展示当前上下文窗口使用百分比 |
| System Foundation 层 | 🔄 | 系统基础指令层，跨会话持久化 |
| Episodic Log 层 | 🔄 | 对话历史日志层，自动滚动摘要 |
| Working Scratchpad 层 | 📋 | 工作记忆层，当前任务临时数据 |
| External Memory 层 | 📋 | 外部记忆层，向量检索相关历史 |
| Isolated Sub-Agents 层 | 📋 | 子 Agent 隔离层，独立上下文执行任务 |
| 工作区记忆 | 📋 | 绑定到工作目录的持久化记忆 |
| 子 Agent 隔离 | 📋 | 工具调用的子 Agent 拥有独立上下文 |
| 上下文手动清理 | ✅ | 手动清除当前会话上下文 |
| 上下文压缩 | 🔄 | 长对话自动压缩历史以节省 Token |
| 相关记忆检索 | 📋 | 自动从历史会话中检索相关上下文 |

### 五层架构概览

```
┌─────────────────────────────────────────┐
│  L5: ISOLATED SUB-AGENTS               │  子 Agent 独立上下文
├─────────────────────────────────────────┤
│  L4: EXTERNAL MEMORY                   │  向量检索的外部记忆
├─────────────────────────────────────────┤
│  L3: WORKING SCRATCHPAD                │  当前任务工作区
├─────────────────────────────────────────┤
│  L2: EPISODIC LOG                      │  对话历史（滚动摘要）
├─────────────────────────────────────────┤
│  L1: SYSTEM FOUNDATION                 │  系统指令 + 工作区记忆
└─────────────────────────────────────────┘
```
